# bodyframe-frontend

React + Vite tabanli BodyFrame arayuzu.

## Kapsam

- JWT giris ve kayit akislari
- E-posta dogrulama
- Sifre sifirlama
- Profil guncelleme ve sifre degistirme
- Kullanici icin olcum hesaplama, kaydetme, gecmis ve grafik
- Doktor icin hasta atama, hasta detayi ve hasta adina olcum girisi

## Gelistirme

1. Bagimliliklari yukleyin:

```bash
npm install
```

2. Gerekirse ortam degiskenini ayarlayin:

```bash
VITE_API_BASE_URL=https://localhost:7156/api
```

3. Gelistirme sunucusunu baslatin:

```bash
npm run dev
```

4. Uretim derlemesi alin:

```bash
npm run build
```
